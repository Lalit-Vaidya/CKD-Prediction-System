# AI-Based Chronic Kidney Disease Prediction System

## 📌 Project Overview
This project predicts Chronic Kidney Disease (CKD) using Machine Learning and generates automated medical reports.

## 🚀 Features
- OCR-based lab report extraction
- CKD prediction using Random Forest
- Feature selection using Firefly & Cuckoo Search
- Automatic PDF medical report generation

## 🛠 Technologies Used
- Python, Flask
- Tesseract OCR
- Random Forest
- MySQL
- ReportLab

## ▶️ How to Run
cd code  
python app.py
